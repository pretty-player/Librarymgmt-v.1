Library Management System
A simple desktop application built with Python and Tkinter to manage book inventory, student circulation, and PDF assets.

Requirements
>Bash
pip install -r requirements.txt
How to Run
Bash
python library_system.py
Default Admin Access
Password: admin123 (Required to view logs or perform a factory reset)

Note: The system automatically creates the local database (library.db), the PDF storage folder (library_pdfs/), and the log file (system_logs.txt) when you run it for the first time.
